# ONE HCMS Learning, Module 09 (LMS), Nabati Group

Paket ini berisi portal LMS multi-persona berbasis HTML standalone, plus kamus kompetensi KSNI.

## Cara pakai
Buka index.html di browser. Login dengan,
- Username, lms_onehcms atau onehcms
- Password, nabatiHC-123!

Pilih salah satu dari 6 persona. Di dalam tiap persona ada switcher untuk berpindah persona, dan toggle bahasa ID atau EN di kanan atas.

## Isi file
- index.html, launcher login dan pemilih persona.
- lms_p_learner.html, Learner (bergaya Coursera), identitas Sari Dewi.
- lms_p_linemanager.html, Line Manager, identitas Budi Santoso.
- lms_p_instructor.html, Instructor dan Content Author, identitas Hendra Kusuma.
- lms_p_ldadmin.html, L&D Admin, identitas Dewi Lestari.
- lms_p_hcbp.html, HR Business Partner, identitas Andi Pratama.
- lms_p_exec.html, L&D Executive, identitas Ratna Wijaya.
- kamus_kompetensi_ksni.html, kamus master 30 fungsi KSNI, 346 kompetensi.
- PROMPT_LANJUTAN.md, prompt untuk melanjutkan pekerjaan di chat baru.

## Catatan
- Semua file standalone, tidak butuh server, cukup dibuka di browser.
- Bilingual ID dan EN, tanpa em dash dan en dash.
- Governance, analitik agregat dengan k-anonymity, tanpa skor risiko individu, maker-checker untuk kontrol sensitif.
- Kompetensi mengikuti Kerangka Leveling Competency KSNI.
